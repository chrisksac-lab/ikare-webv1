# ikare-web
Patient-facing healthcare platform for accessing online consultations and healthcare services through the browser.

## Overview

iKare Web provides a modern web experience for patients seeking healthcare services online.

Users can manage appointments, access healthcare information, consult professionals, and track their medical journey.

## Features

- User authentication
- Appointment booking
- Consultation management
- Patient dashboard
- Medical record access
- Responsive design
- Healthcare recommendations

## Technology Stack

- React
- TailwindCSS
